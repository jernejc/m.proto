# m.proto readme

###### First make sure you have node.js installed
http://nodejs.org/ (this should also install node package manager)

###### Install Bower
npm install -g bower

###### Install Grunt
sudo npm install -g grunt-cli.

###### Clone the repo:
git clone https://github.com/jernejc/m.proto.git

###### Move to the folder
cd m.proto

###### Install dependecies
npm install
bower install

###### Serve with grunt:
grunt serve --f
